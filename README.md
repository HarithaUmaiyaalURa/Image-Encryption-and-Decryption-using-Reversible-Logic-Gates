# Image-Encryption-and-Decryption-using-Reversible-Logic-Gates
Image Encryption and Decryption using Reversible Logic Cryptography Design (RLCD) with LFSR-based key generation. This repository contains project workflow, simulation screenshots, encrypted/decrypted outputs, histogram analysis, and result verification.
**Key Features**
Image encryption and decryption using reversible logic
LFSR-based pseudo-random key generation
Verilog implementation of cryptographic architecture
MATLAB-based image processing and reconstruction
Functional verification through simulation
Histogram-based security analysis
**Reversible Logic Gates Used**
Feynman Gate
Toffoli Gate
Fredkin Gate
SCL Gate
**Results**
The proposed system successfully encrypts image data into an unreadable format and accurately reconstructs the original image during decryption. Histogram analysis demonstrates effective image protection and secure data transmission.
**Applications**
Secure image communication
Hardware cryptographic systems
Low-power VLSI security applications
Embedded security solutions
