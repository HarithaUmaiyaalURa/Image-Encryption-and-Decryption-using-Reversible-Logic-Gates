# Image Encryption and Decryption Using Reversible Logic Gates

## Overview

This project presents a secure image encryption and decryption system based on Reversible Logic Cryptography Design (RLCD).

The system uses a Linear Feedback Shift Register (LFSR) for pseudo-random key generation and reversible logic gates for encryption and decryption operations. The objective is to achieve secure image transmission while leveraging the advantages of reversible computing such as reduced information loss and low-power operation.

---

## Key Features

* Image encryption and decryption using reversible logic gates
* LFSR-based pseudo-random key generation
* Verilog implementation of cryptographic architecture
* MATLAB-based image processing and reconstruction
* Functional verification through simulation
* Histogram-based security analysis

---

## Reversible Logic Gates Used

* Feynman Gate
* Toffoli Gate
* Fredkin Gate
* SCL Gate

---

## Workflow

1. Input image acquisition
2. Binary conversion of image data
3. LFSR key generation
4. Encryption using reversible logic gates
5. Generation of encrypted image
6. Decryption using the same key
7. Reconstruction of the original image
8. Histogram-based result verification

---

## Results

The proposed system successfully encrypts image data into an unreadable format and accurately reconstructs the original image during decryption.

Key observations:

* Secure pseudo-random key generation using LFSR
* Successful encryption and decryption process
* Accurate image recovery
* Improved image security demonstrated through histogram analysis
